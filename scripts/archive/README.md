# Archive

These are old one-off deploy scripts from earlier patch sessions. They have been superseded by the root-level `deploy.js`, which consolidates all command definitions in one place.

| File | What it was |
|------|-------------|
| `deploy-bigpatch.js` | Added `/monsterattack`, `/monsterroll`, `/r` |
| `deploy-downtime.js` | Added the counter-based `/downtime` command |
| `deploy-eberron.js` | Added Eberron setting to `/calendar` and `/weather` |
| `commands-deploy.js` | Added the `/hp` command (was `commands/deploy.js`) |

**Do not run these scripts.** Use `npm run deploy` or `npm run deploy:guild` from the project root instead.
